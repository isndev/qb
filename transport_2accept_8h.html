Detailed Description

TCP connection acceptance transport implementation for the QB IO library.

This file provides a transport implementation for accepting incoming TCP connections, wrapping the qb::io::tcp::listener functionality. It is used by asynchronous acceptor components (e.g., qb::io::async::tcp::acceptor).
