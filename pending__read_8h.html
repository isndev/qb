Detailed Description

Event for notifying about pending (unprocessed) read data in asynchronous I/O.

This file defines the pending_read event structure which is triggered to notify about unprocessed bytes remaining in the read buffer after protocol message processing. Derived classes can handle this event by implementing the void on(pending_read &&) method.
