Detailed Description

Secure (SSL/TLS) TCP connection acceptance transport for the QB IO library.

This file provides a transport implementation for accepting incoming secure (SSL/TLS) TCP connections, wrapping the qb::io::tcp::ssl::listener functionality. It is used by asynchronous secure acceptor components. Requires OpenSSL.
