Detailed Description

template<typename T, std::size_t max_size>
class qb::lockfree::mpsc::ringbuffer< T, max_size, 0 >

Multi-Producer Single-Consumer ring buffer with runtime-determined number of producers.

This specialization provides a MPSC ring buffer with a runtime-determined number of producers specified during construction.

Template Parameters
TThe type of elements stored in the buffer
max_sizeThe maximum capacity per producer buffer
