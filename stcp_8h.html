Detailed Description

Secure TCP (SSL/TLS) stream transport for the QB IO library.

This file provides a transport implementation for secure (SSL/TLS) TCP sockets, extending the qb::io::stream class with qb::io::tcp::ssl::socket specific functionality. It handles encrypted stream-based communication. Requires OpenSSL.
