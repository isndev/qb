Detailed Description

Implementation of UDP sockets for datagram communication in the QB IO library.

This file provides the implementation of UDP sockets supporting both IPv4, IPv6, and Unix sockets (if enabled) for connectionless, datagram-based communication. It builds upon the generic qb::io::socket wrapper.
