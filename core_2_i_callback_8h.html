Detailed Description

Callback interface for the QB Actor Framework.

This file defines the ICallback interface which provides a mechanism for actors to receive periodic callbacks during each VirtualCore loop iteration. This allows actors to perform background tasks, periodic operations, or polling without relying on event-driven behavior.
